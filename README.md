# 🔍 Finvestigate

> Investigate your finances. Track smarter. Spend wiser.

Finvestigate is a personal finance tracker web application 
built with **Python** and **Flask** that helps you monitor 
your income, expenses, and financial habits.

## 🚀 Features (Planned)
- [ ] Track income and expenses
- [ ] Categorize transactions
- [ ] Visual spending reports & charts
- [ ] Monthly budget planning
- [ ] Export financial data

## 🛠️ Tech Stack
